# Security Policy

## Supported Versions

We currently support the latest version of the application with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue on the GitHub repository. We will respond promptly to address the issue.

## Best Practices

- Keep your Google OAuth credentials secure.
- Never commit `.env` files to the repository.
- Use the latest version of dependencies.
- Follow the principle of least privilege for API scopes.
